## Developing

Assuming that <a href="https://nodejs.org/en/">Node</a> is installed on your system.

Install dependencies

```sh

# Install packages
npm install

# Install Typescript compiler
npm install tsc
```

To run a development build

```sh
npm run dev
```

To compile and run in production mode

```sh
# Compile
npm run prodbuild

# Run
npm run start
```
